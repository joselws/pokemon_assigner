# Pokemon Randomizer

## Instructions

Simple program used to get a random Pokemon team based on:

- Team size
- Pokemon Game (Crystal, RSE variants, and Platinum)
- Difficulty

Simply clone the repo and run this command to get started:

```
python3 pokemon_randomizer.py
```
